# The-Unarchiver
Unarchiver一款专注于快速解压和压缩文件的管理工具！这是非常快捷和非常好用的zip、rar文件管理App。

Unarchiver主要功能包括：

- 支持解压Zip, 7z, RAR, Lzh, ZIPX, TAR, GZIP, BZIP, TGZ, TBZ, ISO, DEB, CAB, JAR, RPM, CBS等多种文件。

- 支持多种格式的分卷解压。

- 支持密码解压文件。

- 支持多种格式的文件预览。

- 支持压缩文件的管理和传输。

- 支持系统文件和iCloudDrive访问和导入。

- 支持WiFi传输。

- 支持文件下载功能。
